## Topic scope (per syllabus, tentative order)

1. Introduction to software quality; quality fundamentals; quality culture; software quality requirements; software engineering standards and models
2. Quality & configuration management: reviews, software audits, V&V, SCM, policies/processes/procedures
3. Quality & project management: measurement, risk management, supplier management, SQA plan, software models
4. Quality & software testing: test automation, testing tools, commercial/governmental standards, Pareto principle in SQA, SQA metrics, software testing strategies
